## **b. Données environnementales issues de la télédétection** 

Les données satellitaires sont utilisées pour caractériser les conditions environnementales favorables au développement acridien. - **Végétation** : NDVI, EVI _(dynamique et disponibilité de la biomasse herbacée)issues de_ MODIS (NASA) et Sentinel-2 (ESA) 

- **Température de surface** : LST issue de MODIS (NASA) 

- - **Pluviométrie** : précipitations mensuelles issues de CHIRPS (Climate Hazards Group) 

- Ces données sont utilisées sous forme d’agrégats mensuels, harmonisés spatialement à une résolution d’environ 1 km.
