## _C._ _**Pour l’objectif 3 (spatio-temporelle et technique) :**_ 

- **Zone d’étude** : `o` aire grégarigène du Sud-Ouest (Betioky, Soalara, plateau des Mikea). - **Données utilisées** : `o` séries historiques d’infestations acridiennes (2001–2025), `o` données satellitaires multi-sources (MODIS, Sentinel-2, Landsat), `o` variables environnementales : NDVI, température de surface (LST), pluviométrie. - **Pré-traitement** : `o` extraction des indices environnementaux, `o` harmonisation spatiale et temporelle. - **Modélisation** : `o` algorithme **Random Forest** , `o` génération de cartes de probabilité de présence acridienne. - **Validation** : `o` validation croisée (AUC), `o` confrontation avec observations de terrain disponibles.
