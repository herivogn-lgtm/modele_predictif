## **c. Données climatiques globales** 

La variabilité climatique interannuelle est prise en compte à travers l’indice **ENSO (El Niño / La Niña)** , utilisé comme variable explicative indirecte des anomalies pluviométriques régionales issues de NOAA / American Meteorological Society
