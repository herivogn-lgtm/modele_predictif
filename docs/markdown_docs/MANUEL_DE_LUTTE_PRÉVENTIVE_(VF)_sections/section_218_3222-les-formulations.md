## **3.2.2.2. Les formulations** 

Les matières actives ne sont pas utilisables à l’état pur. Aussi sont-elles formulées avec des adjuvants, solvants et émulgateurs pour les rendre utilisables. Les formulations sont très nombreuses. Elles sont symbolisées par deux lettres qui indiquent le type de formulation : WP pour les poudres mouillables, DP pour les poudres pour poudrages, EC pour les concentrés émulsionnables, UL pour l’ultra-bas volume, etc. 

Quel que soit le type de formulation, la teneur en matière active doit être exprimée en grammes par unité de mesure : en grammes par kilogramme pour les poudres et en grammes par litre pour les liquides. Jamais en pourcentage.
