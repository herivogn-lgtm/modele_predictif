## **1.2.3.2. Action du facteur hydrique** 

En zone tropicale sèche, le facteur hydrique est souvent le principal facteur limitant l’évolution des populations acridiennes. L’eau revêt différentes formes : pluie, rosée, brouillard, vapeur, eau contenue dans le sol. Cette eau exerce une influence directe et/ou indirecte sur le criquet : 

- les effets directs sont particulièrement visibles sur les œufs qui doivent absorber de l’eau très peu de temps après la ponte. Les larves et les ailés recherchent une ambiance hydrique qui leur permette de satisfaire leurs besoins, en buvant par exemple ; 

- les effets indirects sont nombreux et se font par le biais de la végétation consommée et le sol où les criquets pondent.
