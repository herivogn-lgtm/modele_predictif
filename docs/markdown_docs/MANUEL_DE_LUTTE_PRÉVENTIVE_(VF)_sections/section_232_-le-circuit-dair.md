## ! Le circuit d’air 

Le ventilateur centrifuge, en prise directe sur l’axe du moteur, produit un courant d’air à grande vitesse (100 km/h). Une conduite de faible diamètre relie le carter d’air au réservoir de produit, afin de mettre le liquide sous légère pression et d’assurer ainsi un bon écoulement. Le courant d’air produit par le ventilateur est pulsé dans un conduit flexible qui se prolonge par la tuyère au bout de laquelle est fixé le “ Venturi ” où a lieu la rencontre air/produit.
