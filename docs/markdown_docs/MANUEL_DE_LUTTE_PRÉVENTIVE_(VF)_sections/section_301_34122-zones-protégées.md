## **3.4.1.2.2. Zones protégées** 

Les zones protégées sont déterminées avec l'Office national pour l'environnement. Seuls les produits les moins nocifs du point de vue écotoxicologique sont autorisés dans ces zones. Deux types de produits conviennent : les inhibiteurs de croissance contre les larves (exemple : le 

**249** /307 

LA LUTTE ANTIACRIDIENNE 

téflubenzuron) et les chloronicotinyles contre les ailés (exemple : l'imidaclopride).
