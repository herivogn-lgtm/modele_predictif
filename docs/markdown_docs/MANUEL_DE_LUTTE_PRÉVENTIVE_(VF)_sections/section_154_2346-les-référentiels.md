## **2.3.4.6. Les référentiels** 

Les référentiels sont des compilations de données qui permettent de dégager les caractéristiques moyennes de blocs de données géo- et chronoréférencés. Leur principale utilité est de pouvoir confronter chaque situation particulière à une situation moyenne obtenue sur la base du plus grand nombre de répétitions disponibles. 

Cette possibilité est particulièrement intéressante en climatologie quand il s’agit de comparer une situation météorologique à la moyenne, à la médiane ou, mieux, aux normales (valeurs obtenues sur 30 années consécutives). 

Une autre façon de constituer des référentiels est de recourir à l’analyse fréquencielle. Pour cela, on fixe une série de classes significatives pour un phénomène particulier et on calcule le nombre d’occurrences pour un pas de temps donné (décade, mois, trimestre, etc.) sur une période aussi longue que possible. Pour que l’évaluation ait un minimum d’intérêt, il faut compter au moins 10 répétitions, 30 répétitions constituant le minimum statistique. Le résultat de ces analyses se présente sous forme de tableau où les occurrences peuvent être considérées comme une approximation de la probabilité de voir se réaliser un événement donné : une classe pluviométrique pour un mois particulier dans une station donnée. On voit ainsi se dessiner les caractéristiques des saisons des pluies sur une base probabiliste et l’on peut évaluer la variabilité temporelle des événements ou en évaluer la prévisibilité, ce qui n’est pas négligeable pour élaborer les pronostics acridiens. 

Il est bien évident que l’on peut envisager de constituer le même type de référentiel pour les données acridiennes et en particulier sur les densités mensuelles ou décadaires, larvaires ou imaginales.
