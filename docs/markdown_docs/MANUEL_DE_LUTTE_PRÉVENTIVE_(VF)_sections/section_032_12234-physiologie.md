## **1.2.2.3.4. Physiologie** 

Du point de vue du métabolisme général, les grégaires sont plus actifs que les solitaires, ce qui se retrouve dans la vie quotidienne dans les activités de déplacement, de prise de nourriture, respiratoires, etc. Toutefois, la maturation sexuelle est plus rapide chez les solitaires. Chez le Criquet migrateur, elle est au mieux de 11 jours pour les solitaires contre 22,5 jours pour les grégaires. Ceci permet aux solitaires d’accomplir quatre à cinq générations par an alors que les grégaires n’en font que deux ou trois. Les longévités moyennes sont aussi supérieures chez les grégaires, avec 45 jours au lieu de 30.
