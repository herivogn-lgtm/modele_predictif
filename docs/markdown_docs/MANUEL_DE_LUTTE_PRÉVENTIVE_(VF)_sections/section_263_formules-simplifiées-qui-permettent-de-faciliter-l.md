## **Formules simplifiées qui permettent de faciliter les calculs aux techniciens** 

Soit : **Q** pour volume ou quantité à appliquer par ha **Dbt** pour débit en litres par minute **V** pour vitesse d’avancement **Es** pour espacement ou andain **Q** =600 x Dbt / Es x V **Dbt** =V x Es x Q / 600 **V** =600 x Dbt / Es x Q **Es** =600 x Dbt / V x Q
