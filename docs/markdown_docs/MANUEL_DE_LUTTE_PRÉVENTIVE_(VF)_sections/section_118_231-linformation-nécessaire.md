## **2.3.1. L’information nécessaire** 

Le système d’avertissement repose sur l’analyse régulière (décadaire et mensuelle) de trois catégories complémentaires de données : 

- les données météorologiques, 

- les données acridiennes, 

- les données mésologiques.
