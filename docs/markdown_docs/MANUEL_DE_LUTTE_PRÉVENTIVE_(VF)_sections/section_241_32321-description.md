## **3.2.3.2.1. Description** 

Les éléments constitutifs des appareils rotatifs à piles sont le manche tubulaire, le support de tête orientable, le moteur et la source d’énergie électrique, les gicleurs et les disques rotatifs (figure 95). 

**Figure 95. –** Principaux éléments constitutifs des pulvérisateurs à piles (d’après Anonyme (Micron Sprayers), n.d.d).
