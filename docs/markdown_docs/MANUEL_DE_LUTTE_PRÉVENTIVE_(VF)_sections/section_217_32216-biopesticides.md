## **3.2.2.1.6. Biopesticides** 

Les recherches menées à Madagascar sur Metarhizium anisopliae var. acridum et Beauveria spp. n’ont pas encore débouché sur une possibilité d’application pratique. Il reste à préciser les modalités d’utilisation des biopesticides dans le cadre de la lutte préventive, en particulier dans les zones écologiquement sensibles (figure 100).
