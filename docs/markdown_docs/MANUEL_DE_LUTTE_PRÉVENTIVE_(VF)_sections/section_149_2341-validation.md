## **2.3.4.1. Validation** 

Avant toute chose, les données qui parviennent au Service d’avertissement de Betioky sont évaluées, pour vérifier leur intégrité, leur conformité, leurs qualités structurelles et juger de l’opportunité de leur saisie. 

Cette étape est particulièrement importante : 

– pour s’assurer de l’intégrité des données ; 

– pour cerner les incohérences accidentelles ou des dérives systémiques ; 

– pour éviter les saisies multiples d’informations similaires parvenant par des canaux différents ; 

– pour détecter d’éventuelles urgence ou de graves carences de transmission.
