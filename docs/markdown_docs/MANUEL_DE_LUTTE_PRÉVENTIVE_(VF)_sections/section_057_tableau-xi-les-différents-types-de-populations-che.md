Pour les bandes larvaires, voici les caractères pratiques à détailler sur le terrain : 

- dimensions de la bande (du front à l’arrière de la bande) ; 

- longueur et largeur du front de bande ; 

- densité maximale du front de bande ; 

- densité moyenne à l’arrière du front. 

Pour les essaims, voici les caractères pratiques à détailler sur le terrain : 

- dimensions de l’essaim (en ha ou en km²) ; 

- direction (d’où et vers où) ; 

- densité en vol : – faible : essaim visible que s’il est suffisamment proche pour que les criquets soient distingués individuellement (vol clair) ; 

 - moyenne : densité intermédiaire entre faible et forte ; 

 - forte : l’essaim efface certains éléments du paysage ; 

 - très forte : paysage entièrement masqué par l’essaim.
