## **1.3.2. Les phases du Criquet migrateur** 

Le facteur déclenchant la transformation phasaire est la densité. Les effets qui en résultent sont gradués dans le temps : 

- les effets immédiats comme les changements de couleur des criquets et leur comportement ; 

- les effets transmis à la descendance, d’ordre morphologique et physiologique.
