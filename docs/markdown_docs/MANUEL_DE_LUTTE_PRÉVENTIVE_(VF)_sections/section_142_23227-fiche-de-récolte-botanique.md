## **2.3.2.2.7. Fiche de récolte botanique** 

Tout échantillon botanique récolté pour être mis en herbier doit être accompagné d’une étiquette permettant son identification, le chrono- et le géoréférencement de la récolte. Toutes les informations susceptibles de faciliter la détermination doivent figurer sur cette étiquette. 

Un exemple d’étiquette d’herbier est fourni en annexe 7.
