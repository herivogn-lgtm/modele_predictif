## **3.3.1.2.2. Imagos pondants** 

Comme ci-dessus, ce type de cible exige d’être immédiatement stoppé dans sa migration. L’objectif doit être la mortalité des imagos présents mais aussi celle des larves qui vont éclore les jours suivants. Dans ces conditions, ce sont les inhibiteurs de croissance et le fipronil qui conviennent.
