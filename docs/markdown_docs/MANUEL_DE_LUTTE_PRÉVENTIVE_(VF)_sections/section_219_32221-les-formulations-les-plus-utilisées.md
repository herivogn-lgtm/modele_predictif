## **3.2.2.2.1. Les formulations les plus utilisées** 

Les formulations les plus utilisées sont : 

– les poudres pour poudrages (DP) : elles sont peu à peu abandonnées ; 

– les concentrés émulsionnables : liquides concentrés à émulsionner dans l’eau pour pulvérisations aqueuses. Ces formulations sont très peu utilisées ; 

– les formulations huileuses pour traitement UBV (UL) : liquides huileux pour traitements tels quels avec des adjuvants locaux pour avoir des teneurs plus faibles. Les formulations UL sont les plus utilisées.
