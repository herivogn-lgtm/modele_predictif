## **1.3.1.1.3. L’imago** 

L’imago désigne l’insecte ailé qui a achevé son développement larvaire. Le terme d’adulte est réservé à l’imago ayant atteint sa maturation sexuelle. La distinction entre les deux sexes se fait par l’observation de l’extrémité abdominale des insectes (figure 13). 

**Figure 13. –** Reconnaissance du sexe des imagos par l’examen des génitalias externes. 

**ep** : épiprocte ; **pt** : paraprocte ; **cq** : cerques ; **vd, vv, vl** : valves dorsales, ventrales et latérales de l’oviscapte ; **psg** : plaque sous-génitale ; **t7-t11** : tergites ; **s7-s9** : sternites.
