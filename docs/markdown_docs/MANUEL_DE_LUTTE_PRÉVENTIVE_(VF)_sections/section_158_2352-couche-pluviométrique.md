## **2.3.5.2. Couche pluviométrique** 

Les données pluviométriques ponctuelles doivent être converties en surfaces (plages isohyètes) pour être croisées avec la couche des biotopes acridiens. Cette interpolation spatiale demande quelques précautions. Il faut suffisamment de données, correctement réparties dans l’espace : au moins 130 postes pluviométriques sont indispensables pour obtenir des cartes d'isohyètes fiables. La méthode d'interpolation optimale retenue dans le SIG est la méthode d'inverse de la distance (Borot, 2008). À chaque carte d'isohyètes produite, une carte d'incertitude est liée, permettant de vérifier la fiabilité de l'interpolation pluviométrique (figure 66).
