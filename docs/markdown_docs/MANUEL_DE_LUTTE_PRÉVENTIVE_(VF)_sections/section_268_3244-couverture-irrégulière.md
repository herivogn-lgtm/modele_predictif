## **3.2.4.4. Couverture irrégulière** 

Ce type de couverture est en fait intermédiaire ente la couverture totale et le traitement en barrières. Il n’est possible qu’avec les produits qui agissent par ingestion et qui ont une longue persistance d’action. Il est parfaitement indiqué pour l’utilisation des inhibiteurs de croissance sur larves de L2 à L4.
