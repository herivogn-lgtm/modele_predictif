## **1.2.2.3.2. Pigmentation** 

La plupart des larves de locustes en phase solitaire sont de pigmentation uniforme et adaptée à l’environnement, le plus souvent verte ou brune. Les larves grégaires ont une teinte contrastée : un fond jaune plus ou moins orangé, maculé de noir. Les différences pigmentaires sont moins marquées
