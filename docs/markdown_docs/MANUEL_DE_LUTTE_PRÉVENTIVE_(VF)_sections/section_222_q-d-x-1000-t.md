## **Q = d x 1000 / t** 

**Q** en millilitres 

**d** = dose de matière active à utiliser par hectare, en grammes **t** = teneur du produit commercial en grammes (de m. a. par litre) 

Q = 20 x 1000 / 60 = 333 ml de formulation commerciale. 

Il faut donc prendre 333 ml de formulation commerciale et y rajouter 667 ml (1000 - 333 ml) de gazole pour obtenir un litre de mélange.
