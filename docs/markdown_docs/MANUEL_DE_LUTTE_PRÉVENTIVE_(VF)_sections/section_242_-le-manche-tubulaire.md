## ! Le manche tubulaire 

Le manche tubulaire a plusieurs rôles : loger les piles rondes (1,5 volt, type R20) montées en série ; servir, grâce à sa longueur, à faire varier la hauteur d’émission ; manier et transporter le pulvérisateur. À son extrémité est placé l’interrupteur de marche/arrêt.
