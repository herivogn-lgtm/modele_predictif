## **3.2.3.5.2. Calcul des paramètres** 

À Madagascar, ce sont essentiellement les traitements terrestres avec les équipements portables par l’homme et les aéronefs qui sont utilisés en lutte antiacridienne. Les paramètres varient seulement dans les limites imposées par ces deux types de porteurs. 

Pour les traitements avec les appareils terrestres, la vitesse est fixe à 3,6 km/h. Ce sera donc l’espacement qui sera la variable à ajuster en fonction du débit. Pour les aéronefs, tous les paramètres peuvent varier. On choisira toujours l’espacement maximal compatible avec les possibilités du produit et le type de couverture nécessaire. 

Moyens techniques 

**236** 

Avant de commencer les réglages proprement dits, il faut déterminer ce que l’on veut obtenir en fonction du volume d’application, à savoir : quelle quantité par hectare à appliquer ? Quel débit ? Quel espacement ? Quelle vitesse ?
