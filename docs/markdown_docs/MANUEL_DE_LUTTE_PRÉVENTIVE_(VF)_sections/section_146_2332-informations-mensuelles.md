## **2.3.3.2. Informations mensuelles** 

À chaque fin de mois, une synthèse des informations est faite pour être diffusée dans la première décade du mois suivant. Il est alors possible de prendre en compte les résultats des prospections extensives des zones antiacridiennes pour affiner diagnostics et pronostics.
