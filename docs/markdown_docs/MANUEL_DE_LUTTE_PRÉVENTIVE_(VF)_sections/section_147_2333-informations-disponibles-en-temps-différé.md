## **2.3.3.3. Informations disponibles en temps différé** 

Toutes les données qui ne parviennent pas en temps voulu sont néanmoins saisies et archivées, leur utilité sera essentiellement historique car elles alimenteront les référentiels thématiques que le Service d’avertissement se doit de construire. De plus, elles seront utiles pour améliorer les synthèses trimestrielles ou annuelles.
