## **3.4.2.3. Stocks périmés** 

Les stocks périmés proviennent essentiellement d’une mauvaise gestion des entrées et sorties. Il faut donc veiller à ce que la règle “ **premier arrivé = premier sorti** ” soit respectée. 

Cependant, il arrive que l’on reçoive des formulations inutilisables, telles que des produits tellement corrosifs qu’il est impossible de les utiliser tant ils détériorent le matériel ou encore des produits sous forme de mayonnaise. Ces inconvénients peuvent être évités par la rédaction d’un cahier des charges adéquat. 

Le Code de conduite FAO pour la gestion de pesticides prévoit la collaboration des fournisseurs, par l’intermédiaire de l’organisation interprofessionnelle CropLife International, pour le traitement des pesticides périmés. Il convient de regrouper tous les produits ne pouvant plus être utilisés dans le magasin central de Betioky et de prendre contact avec CropLifeMadagascar pour leur traitement.
