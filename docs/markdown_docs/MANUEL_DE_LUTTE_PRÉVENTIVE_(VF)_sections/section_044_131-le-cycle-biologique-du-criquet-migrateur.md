## **1.3.1. Le cycle biologique du Criquet migrateur** 

Comme tout acridien, le Criquet migrateur passe par trois états biologiques : l’ **œuf** , la **larve** et l’ **imago** (figure 11). Contrairement aux sous-espèces de régions tempérées, le Criquet migrateur malgache se développe toute l’année
