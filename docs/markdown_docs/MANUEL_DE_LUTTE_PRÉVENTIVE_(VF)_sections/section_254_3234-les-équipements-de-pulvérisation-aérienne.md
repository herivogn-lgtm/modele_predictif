## **3.2.3.4. Les équipements de pulvérisation aérienne** 

Les interventions antiacridiennes concernent des applications de liquides, dans la grande majorité des cas en UBV. Les volumes appliqués varient généralement de 0,5 à 3 litres par hectare, bien que, dans la plupart des cas, on pulvérise un litre par hectare. Les équipements ci-dessous sont conçus pour appliquer essentiellement des formulations UBV (UL). 

|[a] 

**Figure 96. –** Schéma du système de pulvérisation d’un aéronef. 

**1** : tête de pulvérisation ; **2** : rampe (en trois sections sur les hélicoptères) ; **3** : vanne de commande de débit (O/F) ; **4** : transmission de la commande de pulvérisation ; **5** : régulateur de débit ; **6** : tableau de commande situé dans le cockpit ; **7** : conduite de retour vers la cuve ; **8** : pompe ; **9** : moteur ; **10** : cuve ; **11** : câble de commande du vide-vite ; **12** : fond de cuve ; **13** : position d’ouverture du vide-vite. 

La gestion et l’entretien des composants du système de pulvérisation des aéronefs sont normalement de la compétence des pilotes et des mécaniciens qui les accompagnent. Les paramètres des traitements et les modalités d’application doivent normalement être déterminés d’un commun accord entre le pilote et le responsable du chantier. Ce dernier doit vérifier que les réglages sont conformes aux paramètres choisis.
