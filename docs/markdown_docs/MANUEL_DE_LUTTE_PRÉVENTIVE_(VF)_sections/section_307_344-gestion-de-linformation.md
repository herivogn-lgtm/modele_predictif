## **3.4.4. Gestion de l’information** 

Lors de chaque intervention de lutte, terrestre ou aérienne, une fiche de compte-rendu de traitement doit être remplie. Les données sont géo-, chronoet taxoréférencées. Les caractéristiques de l'application, la description du site, la description de la cible (taxon, phénologie, phase), l'évaluation de l'efficacité comme tous les incidents sont soigneusement mentionnés de façon standardisée (annexe 8). 

Toutes ces informations sont regroupées dans une base de données relationnelles (Microsoft[®] Access) dont l'intérêt est multiple : 

– permettre un suivi objectif de campagne en fonction de divers paramètres : temps, espace, espèces, stades phénologiques, etc. ; 

– permettre une typologie des interventions conduisant à rationaliser la logistique, en particulier les pré-positionnements de produits et de matériels ; 

– compléter les informations acridiennes, en précisant la localisation spatio-temporelle des foyers de grégarisation ; 

– permettre un suivi spatio-temporel à court, moyen et long termes des impacts sanitaire et environnemental des différents types d'application.
