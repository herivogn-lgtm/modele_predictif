## **3.3.2.3. Pâturages** 

Il est indispensable de prévenir les éleveurs du traitement d'une zone, afin d'empêcher les troupeaux d’y pénétrer. Ceci est particulièrement important lorsque les animaux produisent du lait pour la consommation humaine, les matières actives risquant de se retrouver concentrées dans le lait.
