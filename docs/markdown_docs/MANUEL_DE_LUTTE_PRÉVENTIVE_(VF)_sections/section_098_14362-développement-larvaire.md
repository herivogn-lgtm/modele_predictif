## **1.4.3.6.2. Développement larvaire** 

La vitesse de développement dépend essentiellement de la température. La température minimale moyenne pour le développement larvaire est de 26,7 ºC. Le Criquet nomade est nettement plus thermophile que le Criquet migrateur : l'optimum thermique du développement larvaire se situerait à 33 ºC, contre 25ºC pour les larves de Locusta. 

Les larves ne survivent pas au-delà du 2[e] stade si la température est inférieure à 22 ºC. Une forte humidité est nécessaire (l'optimum se situerait entre 50 à 75 % d'humidité). Les faibles valeurs d'humidité retardant le développement (30 % d'humidité à 27 ºC) sont très proches des valeurs létales par dessiccation. Inversement, les taux d'humidité supérieurs à l'optimum provoquent un retard progressif du développement, jusqu'à l'empêcher. Au delà du 1[er] stade, les larves résistent mieux aux températures et aux humidités extrêmes mais une séquence sèche dans le développement est toujours préjudiciable.
