## **3.1.2. Les stratégies d’intervention** 

Si la stratégie préventive est ordinairement privilégiée, elle n’est, malheureusement, pas toujours mise en œuvre avec suffisamment de détermination ou de persévérance et la loi naturelle s’impose : dès que les conditions écométéorologiques sont favorables, le processus de recrudescence s’enclenche. Il faut appliquer d’autres stratégies pour tenter d’endiguer le nouveau fléau acridien ; mieux vaut alors disposer d’un plan de gestion programmée des risques.
