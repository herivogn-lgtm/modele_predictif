## **2.3.4.3. Structure de la base de données acridiennes** 

La base de données acridiennes, dans sa version simplifiée, est constituée de six tables (figure 62) : 

– la table **Relevé** : table principale, elle assure le chronoréférencement et est liée aux tables secondaires ; 

– la table **Station** : pour le géoréférencement et les composantes statiques de l’environnement ; 

– la table **List_prospecteur** : pour l’identité et les caractéristiques de l’observateur ; 

– la table **Environnement** : pour les composantes dynamiques de l’environnement ; 

– la table **LMC** : pour les données de structure de population du Criquet migrateur malgache ; 

– la table **NSE** : pour les données de structure de population du Criquet nomade. 

À ces six tables, on pourrait adjoindre deux tables d’informations synthétiques, l’une pour Locusta, l’autre pour Nomadacris. 

Ces tables sont assorties de “requêtes” qui permettent d’extraire les informations, à la demande, en fonction de la structure d’état de sortie préétablie.
