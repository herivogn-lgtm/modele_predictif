## **3.3.4.4. Productivité : coût temporel de l’hectare traité** 

Toute intervention se doit d'être productive, c'est-à-dire d'obtenir le maximum de résultats en faisant intervenir le minimum de moyens (financiers, temps passé). Pour être efficace, il faut agir vite et bien, d'où la mise au point de techniques particulières, comme l'application en barrières et en UBV de pesticides sur les bandes larvaires. Il importe de protéger le maximum de surface dans le minimum de temps.
