## **2.3.4.5. Base de données des traitements** 

Parallèlement aux bases de données acridiennes et météorologiques, le CNA devrait se doter d’une base de données des traitements, regroupant toutes les informations concernant les opérations de lutte : chaque chantier de lutte donnant lieu à une fiche d’information chrono-, géo- et taxoréférencée. Outre l’intérêt de pouvoir, à terme, procéder à une typologie des interventions de lutte, cette base de données permettrait de compléter les signalisations de la base acridiennes puisque toute intervention de lutte est censée concerner une pullulation acridienne de plus ou moins grande étendue. Enfin, une telle base de données est indispensable pour effectuer un suivi de l’impact des traitements acridicides.
