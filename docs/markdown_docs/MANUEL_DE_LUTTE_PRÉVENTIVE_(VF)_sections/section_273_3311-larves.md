## **3.3.1.1. Larves** 

Les larves peuvent se présenter sous différents types de cible. On peut rencontrer des taches larvaires plus ou moins denses, plus ou moins grandes. La tactique à utiliser dépend de la forme de la cible mais aussi du mode d’action du produit.
